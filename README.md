
# RMap - Reconnaissance Mapper

## Install

### PyPI

```
$ pip3 install python-rmap
```

### Git

```
$ git clone https://github.com/syspuke/rmap.git
$ cd rmap
$ pip3 install -e .
```

## Usage

```
$ sudo rmap -h
usage: rmap [-h] --ip IP [-d]

options:
  -h, --help  show this help message and exit
  --ip IP     IP Address
  -d          Debug output
```
